## Endpoints

### GET /api/scores

- **Description**: Retrieves the current high scores.
- **Parameters**: None
- **Response**: Returns a JSON object with high scores.

### POST /api/score

- **Description**: Submits a new score.
- **Parameters**: 
  - `score` (integer): The score to submit.
- **Response**: Returns a confirmation message.
